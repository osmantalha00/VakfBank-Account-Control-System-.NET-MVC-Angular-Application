# VakıfBank Account Control System
staj proje

This project is a full-stack banking application developed using .NET MVC for the backend and Angular for the frontend.
It provides a comprehensive account management interface that allows users to view, create, update, and delete customer accounts and transactions.

The system uses MongoDB for data storage and includes Swagger integration for API testing and documentation.
Each transaction automatically updates the account balance, and all operations are recorded on the Transaction History page.

Key Features:

* Client–Server architecture built with .NET MVC and Angular

* CRUD operations for customers, accounts, and transactions

* Real-time balance updates after each transaction

* MongoDB integration for fast and flexible data storage

* Swagger UI for API testing and documentation
